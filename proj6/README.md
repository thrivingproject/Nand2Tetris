# Hack Language Assembler

An assembler that translates programs written in Hack assembly language (symbolic machine code) into binary machine code according to the Hack instruction set.

## Usage

Where `Prog.asm` is a text file containing Hack assembly

```shell
py HackAssembler.py [path/to/]Prog.asm
```
